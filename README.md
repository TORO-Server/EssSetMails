# EssSetMails

Minecraftのプラグイン"Essentialsx"のメールの数を指定した数にするためのスクリプト

## 使い方

下記のコマンドを実行すると result.yml が生成されます。

```rb
ruby main.rb メッセージ 送信元プレイヤー名 送信元プレイヤーUUID 送られた時間 メールの個数
```

### コマンドの説明

#### メッセージ

送りたいメールのメッセージを入力してください

#### 送信元プレイヤー名

メールを送ったプレイヤーの名前

#### 送信元プレイヤーUUID

メールを送ったプレイヤーのUUID

#### 送られた時間

タイムスタンプ(単位はミリ秒) の数値で設定してください。

省略した場合は現在の時間が設定されます。

#### メールの個数

省略した場合は 1 が設定されます。
